
# bsc-portfolio

## How to run

 1. `yarn install`
 2. `export WALLET="0x1231231231abcdefg"`  your metamask wallet
 3. `node src/index.js`

## Support protocols

 1. [Mars Ecosystem](https://app.marsecosystem.com/farms)
 2. [gyro.money](https://gyro.money/)


** If your intested to deploy on Kubernetes visit `k8s` folder
`kubectl apply -f k8s`
